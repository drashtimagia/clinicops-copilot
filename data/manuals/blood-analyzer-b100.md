# Blood Analyzer B100 — Quick Service Manual

## Device Overview
The Blood Analyzer B100 is used for rapid sample-based analysis including routine blood chemistry and panel testing.

## Common Use Areas
- Lab 1
- Lab 2
- Pathology

## Common Symptoms
- Sample rejected
- Calibration failed
- Analyzer not reading cartridge
- Delayed results
- Temperature error
- Repeated internal error alerts

## Error Codes
### E301 — Sample Error
Sample volume or integrity is invalid.

**Action**
1. Confirm correct sample type.
2. Check sample volume.
3. Recollect if contaminated or insufficient.

### E302 — Cartridge Not Detected
Consumable is not seated or unreadable.

**Action**
1. Remove and reinsert cartridge.
2. Check expiration date.
3. Replace damaged cartridge.

### E303 — Calibration Failed
Calibration did not complete successfully.

**Action**
1. Restart analyzer.
2. Use approved calibration material.
3. Retry calibration once.
4. Escalate if failure repeats.

### E304 — Temperature Out of Range
Internal conditions are outside required operating range.

**Action**
1. Check room temperature.
2. Ensure ventilation is not blocked.
3. Allow analyzer to stabilize.
4. Retry after 10 minutes.

### E305 — Internal Processing Error
Analyzer encountered a system fault during processing.

**Action**
1. Stop current run.
2. Restart analyzer.
3. Retry with fresh consumable.
4. Escalate if repeated.

## Troubleshooting

### Sample Keeps Getting Rejected
1. Confirm proper collection method.
2. Inspect sample quality.
3. Verify cartridge compatibility.
4. Retry with new sample.

### Calibration Fails Repeatedly
1. Confirm correct calibration kit.
2. Clean external cartridge bay if needed.
3. Power cycle analyzer.
4. Escalate after second failure.

### Delayed Result Generation
1. Check analyzer workload.
2. Restart pending run.
3. Inspect for temperature alerts.
4. Move urgent testing to alternate analyzer if available.

## Safety Notes
- Do not report results from failed calibration runs.
- Suspected sample contamination must follow sample-contamination SOP.
- Repeated internal errors require device quarantine.

## Maintenance
- Clean sample area each shift
- Verify calibration schedule daily
- Inspect cartridge bay weekly
- Full service quarterly
