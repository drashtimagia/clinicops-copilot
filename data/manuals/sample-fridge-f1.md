# Sample Fridge F1 — Quick Service Manual

## Device Overview
The Sample Fridge F1 stores temperature-sensitive clinical samples and reagents under controlled refrigeration.

## Common Use Areas
- Lab 1
- Lab 2
- Storage
- Pathology

## Common Symptoms
- Temperature too high
- Door alarm active
- Cooling failure
- Power loss recovery
- Frost buildup
- Display not responding

## Error Codes
### E401 — High Temperature Alert
Internal temperature exceeds safe range.

**Action**
1. Keep door closed.
2. Confirm recent door opening frequency.
3. Move high-priority samples if alternate cold storage is available.
4. Escalate if temperature does not recover quickly.

### E402 — Door Open Alarm
Door is not sealed properly.

**Action**
1. Close door fully.
2. Check gasket obstruction.
3. Confirm alarm clears.

### E403 — Sensor Fault
Temperature sensor reading is invalid.

**Action**
1. Restart fridge display panel.
2. Compare with backup thermometer if available.
3. Quarantine contents if temperature cannot be verified.

### E404 — Compressor Fault
Cooling subsystem may not be functioning.

**Action**
1. Do not overload unit.
2. Check power connection.
3. Move critical samples to alternate storage.
4. Escalate immediately.

### E405 — Power Recovery Event
Recent power interruption detected.

**Action**
1. Review current internal temperature.
2. Verify sample viability per policy.
3. Document downtime event if threshold exceeded.

## Troubleshooting

### Fridge Is Warm
1. Confirm power supply.
2. Check whether door is sealing correctly.
3. Reduce door opening.
4. Move critical samples if temperature remains high.

### Alarm Keeps Repeating
1. Identify error code.
2. Check door and airflow.
3. Verify room ventilation.
4. Escalate if unresolved.

### Display Is Blank
1. Confirm outlet power.
2. Check breaker or power strip.
3. Restart display if power is present.
4. Escalate if unit remains unresponsive.

## Safety Notes
- Sample integrity takes priority over device troubleshooting.
- Move critical material early if cooling is unstable.
- Follow power outage and sample contamination SOPs where relevant.

## Maintenance
- Temperature log each shift
- Door seal inspection weekly
- Interior defrost/cleaning monthly
- Sensor verification monthly
