# Infusion Pump X2 — Quick Service Manual

## Device Overview
The Infusion Pump X2 is used for controlled intravenous fluid and medication delivery. It supports continuous infusion, rate-based delivery, and alarm-based interruption.

## Common Use Areas
- ICU
- ER
- Room 101
- Room 102

## Common Symptoms
- Pump not powering on
- Battery drains too quickly
- Occlusion alarm
- Air-in-line alarm
- Flow stopped unexpectedly
- Keypad unresponsive

## Error Codes
### E101 — Occlusion Detected
Fluid flow is blocked downstream.

**Action**
1. Check tubing for kinks or clamps.
2. Inspect IV line for blockage.
3. Ensure tubing is seated correctly.
4. Restart infusion only after blockage is cleared.

### E102 — Air in Line
Air detected in infusion tubing.

**Action**
1. Stop infusion immediately.
2. Inspect tubing for bubbles.
3. Prime line again if needed.
4. Resume only when line is air-free.

### E103 — Low Battery
Battery charge is below operating threshold.

**Action**
1. Connect to AC power.
2. Confirm charging indicator is active.
3. If battery does not recover, escalate for battery replacement.

### E104 — Door Open
Pump door is not fully latched.

**Action**
1. Re-seat tubing.
2. Close and lock door firmly.
3. Restart pump.

### E105 — Flow Error
Expected flow rate is not being maintained.

**Action**
1. Check programmed rate.
2. Confirm tubing compatibility.
3. Inspect cassette and line path.
4. Restart device once.

## Troubleshooting

### Pump Will Not Start
1. Confirm battery charge or wall power connection.
2. Hold power button for 3 seconds.
3. Check for visible connector damage.
4. If still unresponsive, quarantine device.

### Frequent Occlusion Alarm
1. Check patient-side tubing path.
2. Confirm clamp is open.
3. Replace tubing set if needed.
4. Retry infusion.

### Battery Not Holding Charge
1. Fully charge the pump.
2. Test off power for 5 minutes.
3. If battery drains rapidly again, escalate to biomedical technician.

### Keypad Not Responding
1. Power cycle device.
2. Clean button surface with approved wipe.
3. If input remains delayed or frozen, remove from service.

## Safety Notes
- Never ignore air-in-line alarms.
- Do not continue infusion if rate accuracy is uncertain.
- Escalate repeated battery, flow, or sensor faults.

## Maintenance
- Clean exterior after each shift
- Inspect tubing door latch daily
- Battery health check monthly
- Calibration review every 6 months
