# SOP-013 — Sample Contamination Response

## Purpose
Provide a standard response when a clinical sample is suspected to be contaminated, mislabeled, leaked, or otherwise compromised.

## Trigger Conditions
Use this SOP when:
- sample container leaks
- sample is mislabeled
- sample integrity is uncertain
- analyzer rejects sample due to quality concerns
- visible contamination is present

## Immediate Actions
1. Stop processing the affected sample.
2. Isolate the sample from other materials.
3. Mark sample as "Do Not Process".
4. Notify the responsible lab technician or supervisor.

## Safety Steps
1. Wear appropriate PPE.
2. Treat the sample as potentially hazardous.
3. Clean affected surfaces using approved disinfectant.
4. Dispose of contaminated consumables per protocol.

## Documentation
Record:
- time of discovery
- sample type
- source room
- handler name
- contamination type
- whether recollection is required

## Recollection Rules
Recollect the sample if:
- labeling is unclear
- sample volume is compromised
- container seal failed
- contamination affects test validity

## Escalation
Escalate to supervisor if:
- multiple samples are affected
- contamination source is unknown
- patient-critical turnaround is impacted

## Communication
Suggested staff notification:
"Sample integrity issue detected. Processing paused pending recollection or supervisor review."
